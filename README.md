# AI Research Starter

这是我的 AI 学习入门项目，用来练习 Python、数据分析和机器学习。

This is my starter project for learning Python, data analysis, and machine learning.

## Project Goal / 项目目标

The goal is to practice a complete AI project workflow.

这个项目的目标是练习一个完整的 AI 项目流程：

1. Organize data, code, notebooks, and outputs.
   整理数据、代码、实验笔记和输出结果。
2. Analyze CSV data with Python and pandas.
   使用 Python 和 pandas 分析 CSV 数据。
3. Visualize results with matplotlib.
   使用 matplotlib 把结果画成图。
4. Train a simple machine learning model with scikit-learn.
   使用 scikit-learn 训练一个简单的机器学习模型。
5. Save progress with Git and GitHub.
   使用 Git 和 GitHub 保存学习进度。

## Project Structure / 项目结构

- `data/`: Input datasets used by the scripts.
  存放输入数据，比如 CSV 数据集。
- `src/`: Python source code.
  存放 Python 源代码。
- `notebooks/`: Jupyter notebooks for future experiments.
  存放以后用于实验和记录的 Jupyter Notebook。
- `outputs/`: Generated reports, charts, and result files.
  存放程序生成的报告、图表和结果文件。
- `requirements.txt`: Python packages required by this project.
  记录项目需要安装的 Python 第三方库。
- `.gitignore`: Files and folders that should not be tracked by Git.
  告诉 Git 哪些文件或文件夹不需要提交。

## Data Files / 数据文件

- `data/students.csv`: Student names and scores for grade analysis.
  学生姓名和成绩数据，用于成绩分析。
- `data/study_scores.csv`: Study hours and exam scores for score prediction.
  学习时间和考试分数数据，用于分数预测模型。

## Python Scripts / Python 脚本

- `src/hello_ai.py`: First Python test script.
  第一个 Python 测试脚本。
- `src/grade_analysis.py`: Uses Python's built-in `csv` module to analyze student scores.
  使用 Python 自带的 `csv` 模块分析学生成绩。
- `src/grade_analysis_pandas.py`: Uses pandas to analyze scores, generate rankings, and create a bar chart.
  使用 pandas 分析成绩、生成排名，并用 matplotlib 生成柱状图。
- `src/score_prediction.py`: Trains a linear regression model to predict exam scores from study hours.
  训练一个线性回归模型，根据学习时间预测考试分数。

## How to Install Dependencies / 如何安装依赖

Install the required Python packages.

安装项目需要的 Python 库：

```powershell
python -m pip install -r requirements.txt
```

## How to Run / 如何运行

Run the pandas grade analysis.

运行 pandas 成绩分析脚本：

```powershell
python src/grade_analysis_pandas.py
```

Run the score prediction model.

运行分数预测模型：

```powershell
python src/score_prediction.py
```

程序可以在 `outputs/` 文件夹中生成这些结果：

这些输出文件可以通过代码重新生成，所以可能不会被 Git 跟踪。
