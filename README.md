# AI Research Starter

This folder is a starter workspace for AI research, experiments, and notes.

## Structure

- `data/`: Store datasets or small sample data files used by experiments.
- `src/`: Store reusable source code, scripts, modules, and utilities.
- `notebooks/`: Store Jupyter notebooks for exploration, analysis, and prototypes.
- `outputs/`: Store generated results such as charts, model outputs, reports, and logs.
- `README.md`: Explain what the project is and how the folders are organized.
- `.gitignore`: Tell Git which local files should not be tracked.
